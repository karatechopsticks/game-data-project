Kinect Rush: A Disney-Pixar Adventure is a platform video game based on Disney Pixar films, released for Kinect on Xbox 360. Announced on 8 March 2012,[1] the game is similar to Kinect Disneyland Adventures, but players instead are taken through the worlds of eight of Pixar's movies: Up, Toy Story, Toy Story 2, Toy Story 3 The Incredibles, Cars, Cars 2, and Ratatouille with the game hub set in a theme park.[2] It is currently the last Disney/Pixar videogame to be developed by Asobo Studio.

Reception[edit]

Review aggregator Metacritic gave the game a rating of 68, which indicates "mixed or average reviews".[3]
